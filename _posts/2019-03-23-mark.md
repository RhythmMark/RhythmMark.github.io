## ssh免密登陆场合

一般使用ssh，都是``ssh user@host``然后输入密码

当然现在我更常用的是``免密登陆``

### 0x01 查看（client端）公钥

``cat ~/.ssh/id_rsa.pub``

### 0x02  （在sever端）添加公钥

``vim ~/.ssh/authorized_keys``

把公钥追加上去即可，保存后，拥有该公钥的客户端就有登陆该服务器的权限了。

此时``ssh -p 端口 用户名@服务器域名或ip``即可登录服务器。

接下来是能让以后登录更方便的设置

###0x03

``vim ~/.ssh/config``

增加

```
Host 自取别名

User 同户名

HostName 服务器域名或ip

Port 端口号

IdentityFile ~/.ssh/id_rsa
```

以后要登陆服务器直接``ssh 自取别名``即可
